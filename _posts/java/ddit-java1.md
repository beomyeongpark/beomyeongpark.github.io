# `데이터, 연산`

## 데이터
- 기본형
    - 숫자, 문자, 논리
- 배열
    - 같은 것끼리 묶어서 이용하는것
- 클래스
    - 다른 것끼리 묶어서 사용하는것
    
## 연산
- 산술연산
    - +,-,*,/,%    
- 비교연산  
    - >, <, >=, <=, ==, !=
- 논리연산
    - &&,||,!
- 조건문
    - if, switch
- 반복문
    - for,while,do-while   
    
## 기본형 타입
- 정수
    - byte(1), short(2), int(4), long(8)
- 실수 
    - float(4), double(8)
- 문자 
    - char(2)
- 논리
    - boolean(1)
